# FF-IV 디스코드 챗봇 🐸
author : toto9602<br>
Last Updated : 2024.05.10

> [파이널판타지14] (이하 FF-IV) 관련 퀴즈를 출제하는 챗봇입니다. 🙃 

## Tech Stack
- Spring Boot, Java, Docker & Docker-compose
- JDA(Java Discord API) & JDA-utilities

## Available Commands
### / 설명
- 봇이 지원하는 명령어 목록을 안내합니다.

### /잡 {questionCount}
- FF-IV 인게임 내 [ 잡 ] 관련 문항을 questionCount 개만큼 출제합니다.

## Core Dependencies

- JDA : 5.0.0-alpha.11
- JDA-utilities : 3.0.5
